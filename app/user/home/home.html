Welcome to XMRpool.net! A high performance pool that has nodes in several key locations around the globe and exremely low fees. We promise to bring you the best ping, uptime and returns no matter where your based. Join now & help strengthen the XMR network while creating some XMR on the way!

Start mining now (visit getting started) or track your rigs at the Dashboard

Please stay tuned for more features and changes! If you have a feature request, please poke Snipa on irc.freenode.net in #monero-pools

Pool Features!

  • PaymentID payments for Exchanges
  • PPLNS and Solo mining systems
  • Patched for the 127 transaction bug
  • Geographically diverse server locations with GeoDNS
  • Well-documented API with active development efforts
  • Payments direct to BTC via Shapeshift.io and XMR.to depending on who's got the best rates
  • Supports the Monero network by offering high speed block transit from London to Singapore via the USA
  • Set your own payment thresholds for normal wallets + exchanges!
  • Per-Worker hashrate charts
  • Payments every 2 hours
  • E-mail/SMS notification of miners down

New Features!

  • 12/31/2017 - New Username/Password Formats (Check the FAQ for details!)
  • 12/29/2017 - Integrated Address Support (Tux Exchange users rejoice!)

Upcoming Features

  • Websocket based stats for rapid updates
  • New stratum protocol for XMR as developed by Wolf0
